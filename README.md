# Visualize Excellence
[Swim3D](https://rene78.github.io/Swim3D/) is an interactive visualization of Australian Olympian Dan Smith’s freestyle stroke. Designed for learners and athletes alike, it breaks down the complexities of the stroke cycle into a clean, easy-to-study 3D experience.
